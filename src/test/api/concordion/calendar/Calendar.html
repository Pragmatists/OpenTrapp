Example 1: Displaying particular month

  • When GET request is being send to: /endpoints/v1/calendar/2014
  • Then response status should be equal to HTTP/1.1 200 OK
  • And response should look like: 
    {
    "days": [
        {
            "link": "/endpoints/v1/calendar/2014/01/01",
            "id": "2014/01/01",
            "holiday": true
        },
        // some items...
        {
            "link": "/endpoints/v1/calendar/2014/01/02",
            "id": "2014/01/02",
            "holiday": false
        }
    ]
}

Example 3: Displaying links to next and previous year

  • When GET request is being send to: /endpoints/v1/calendar/2014
  • Then response status should be equal to HTTP/1.1 200 OK
  • And response should look like: 
    {
    "link": "/endpoints/v1/calendar/2014",
    "next": "/endpoints/v1/calendar/2015",
    "prev": "/endpoints/v1/calendar/2013",
    "months": [
        // months data
    ]
}

Example 4: Displaying links to next and previous month

  • When GET request is being send to: /endpoints/v1/calendar/2014
  • Then response status should be equal to HTTP/1.1 200 OK
  • And response should look like: 
    {
    "link": "/endpoints/v1/calendar/2014/01",
    "next": "/endpoints/v1/calendar/2014/02",
    "prev": "/endpoints/v1/calendar/2013/12",
    "days": [
        // days data
    ]
}
